# Transport-system-
This project is a transport system web application that allows users to:

- Create transport routes with origin, destination, base price, and distance.
- Generate tickets for passengers linked to routes.
- Automatically calculate ticket prices based on current weather conditions.
- List all routes and tickets dynamically on the frontend.

Built with Node.js, Express, MySQL, and a simple HTML/CSS/JS frontend. CORS enabled for seamless frontend-backend communication.
